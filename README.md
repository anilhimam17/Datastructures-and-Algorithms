# Datastructure-and-Algorithms
This repository consist of an exhaustive set of implementations composed of data structures and algorithms at the foundation of computer science and software development in C++ and Python.

### Current Status
While the repository is divided among the two main module:
- C++ Implementations
- Python Implementations

The Python Implementations module is currently the most complete off the two and contains many fundamental datastructures and algorithms as follows:
- **Linear Structures:** arrays, linked lists, stack, queue.
- **Non Linear Structures:** btrees, binary search trees, avl trees, heaps, tries, hashing and graphs.
- **Searching Algorithms:** linear search, binary search and graph traversal algorithms (BFS, DFS, Single Source Shortest Path, All Pair Shortest Path, Minimum Spanning Tree).
- **Sorting Algorithms:** bubble sort, selection sort, insertion sort, bucket sort, merge sort, heap sort and quick sort.

In a couple of week I am working towards implementing each of the above mentioned algorithms and datastructures within the C++ Implementaions module to complete the repository.
